# 20260107 首页输入框视觉增强与体验优化

## 用户要求
放大首页输入框的效果，突显“通过输入记录内容”的项目主题。

## 工作过程
1. **尺寸与排版放大**：
   - 将输入框字体大小从 `text-sm` 提升至 `text-lg`。
   - 增加了 `min-h-[80px]` 和更大的内边距，使输入体验更加开阔。
   - 支持高度自适应（Auto-resize），随内容增多自动向下延伸，最高可达 `60px`（约 10 行）。

2. **交互动效增强**：
   - **动态反馈**：当用户开始输入时，容器会触发 `scale-[1.02]` 的放大效果。
   - **阴影与边框**：输入状态下增加了更深邃的投影（`shadow-[0_20px_50px_rgba(0,0,0,0.1)]`）和外发光（`ring-4 ring-slate-50`），使其在视觉上更具浮动感和层级感。
   - **文字计数**：右下角新增了隐秘的字符计数器，仅在有内容时显示，增加记录的仪式感。

3. **语音输入 (STT) 视觉同步**：
   - 同样放大了语音录制状态下的卡片尺寸和文字大小（`text-2xl`）。
   - 增强了呼吸灯（Pulse）和波纹（Ping）动画，突显“聆听”状态。

4. **整体布局优化**：
   - 在 `page.tsx` 中增加了输入框与下方时间轴之间的间距（从 `gap-6` 增加到 `gap-12`），为放大的输入框留出足够的视觉呼吸空间。
